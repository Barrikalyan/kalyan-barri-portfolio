## Color Scheme

### Primary Colors
- **Blue**: `from-blue-600 to-cyan-500` (buttons, headers)
- **Green**: `from-green-600 to-emerald-500` (authenticated state)
- **Red**: `bg-red-600` (danger/delete actions)
- **Amber**: `bg-amber-600` (featured/highlights)

### Background Colors
- **Panel**: `glass-panel` (glassmorphic with transparency)
- **Input**: `bg-white/5 dark:bg-black/20`
- **Overlay**: `bg-black/50` (dark backdrop)

### Text Colors
- **Primary**: White text on dark backgrounds
- **Secondary**: `text-[color:var(--muted)]` (gray/dimmed)
- **Links**: `text-blue-400` (interactive elements)

## Animation States

### Buttons
```
Idle:     Normal size

Active:   scale-95 (press effect)
Disabled: opacity-50
```

### Modals
```
Enter:    scale: 0.9 вҶ’ 1.0, opacity: 0 вҶ’ 1
Exit:     scale: 1.0 вҶ’ 0.9, opacity: 1 вҶ’ 0
```

### Content
```
Tab Change:  opacity: 0 вҶ’ 1, y: 20 вҶ’ 0
Item Add:    opacity: 0 вҶ’ 1 (fade in)
```
